# Legacy Archive Landing Page

A beautiful, high-converting landing page for **Legacy Archive** — the app that helps parents effortlessly capture daily life and securely pass it on to their children with full user-controlled storage.

## Features
- Emotional hero section with Eternal Tree branding
- Clear explanation of user-controlled storage options
- Simple waitlist signup form
- Mobile-responsive design (Tailwind CSS)
- Smooth scrolling and modern UX

## Deployment on GitHub Pages (Recommended)

### Step-by-step:

1. **Create a new GitHub Repository**
   - Go to [github.com/new](https://github.com/new)
   - Name it `legacy-archive` (or similar)
   - Make it **Public**
   - Initialize with a README (optional)

2. **Upload the files**
   - Upload `index.html`
   - Upload `script.js`
   - (Optional) Upload any logo images to an `assets/` folder

3. **Enable GitHub Pages**
   - Go to **Settings** → **Pages** (left sidebar)
   - Under "Source", select **Deploy from a branch**
   - Choose `main` branch → **Save**

4. **Wait 1-2 minutes**
   - Your site will be live at: `https://YOURUSERNAME.github.io/legacy-archive`

5. **Custom Domain (Optional)**
   - Buy `legacyarchive.app` or similar
   - Add it in GitHub Pages settings
